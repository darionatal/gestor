
# Projeto Python - Hello World

Este é um projeto inicial em Python com acesso ao banco de dados PostgreSQL.

## Como executar

1. Certifique-se de ter o Python instalado.
2. Execute o arquivo principal:

```bash
python main.py
```

## Testar conexão com o banco de dados PostgreSQL

O projeto inclui um script para testar a conexão com o banco de dados `cupom_db`.

**Configurações padrão:**
- Banco: `cupom_db`
- Usuário: `postgres`
- Senha: `123`
- Host: `localhost`
- Porta: `5432`

Para testar a conexão, execute:

```bash
python db_test.py
```
